❓ คิดก่อนกดต่อ
computeRoutes รับ travelMode ค่าเดียวต่อ 1 route. วันนี้ Leg 1–2 เป็น Drive แต่ Leg 3 เป็น Walk — ถ้ายิง ทั้งวันในคอลเดียว (per-day) ต้องเลือก mode เดียว แล้ว Leg 3 (Walk) จะถูกคิดเป็นอะไร?
⚠️ per-day mode เดียว → Leg 3 เพี้ยน
ถ้า per-day ส่ง DRIVE ทั้ง route: Leg 3 ที่ควร เดิน 0.3 กม. ข้ามสะพานคนเดิน จะถูกคิดเป็น ขับรถอ้อมหลายกิโล (รถข้ามสะพานคนเดินไม่ได้) → ระยะ + เวลา + เส้นทางผิด. วัน mixed-mode จึงต้องแตกเป็น per-leg อยู่ดี.
✅ สรุป — เลือก per-leg
✓ mode-safe: Leg 3 ได้ระยะ เดินจริง · ✓ reuse cache เดิม (แก้ Stop เดียวไม่ miss ทั้งวัน) · ✓ ไม่ต้องรื้อ handler · ✓ ต้นทุน $0 ใน free tier
Trade-off: วัน cold-cache = N call — แต่ cache 12 ชม. + free tier ซับได้. per-day batching เก็บเป็น Phase 2 เฉพาะวัน single-mode.
