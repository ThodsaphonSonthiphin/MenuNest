#36 — เพิ่ม "สถานที่ใหม่" + ลิงก์รีวิว (TikTok) จากปุ่มเพิ่มจุดแวะในหน้าแผนเที่ยว

จากชีต "เลือกจุดแวะ" มีทางลัด + เพิ่มสถานที่ใหม่ → สลับเข้าโหมดจับสถานที่แบบแผนที่เต็ม (ค้นหา / แตะหมุด / วางลิงก์) → การ์ดพรีวิวมี ช่องลิงก์รีวิว (TikTok) เพิ่มเข้ามา → กด "เพิ่มเป็นจุดแวะ" แล้วจับ 1 ที่ + เพิ่มเป็นจุดแวะในวันนั้น + เด้งกลับแผนเที่ยวทันที (ช่องลิงก์รีวิวนี้เป็นการ์ด Capture ตัวเดียวกับที่ใช้ในแท็บคลังสถานที่ ⇒ จับที่ใหม่จากตรงไหนก็แนบลิงก์รีวิวได้)
