Feature - Place checklist "สิ่งที่ต้องเตรียม" - issue #23 - CHOSEN design
A Checklist item is a User-scoped, reusable label (ADR-058) - define once, reuse it on any place in any trip. Attaching it to a place creates a Place checklist entry that carries a per-place checkbox (ADR-059). Phase 1 lives only in this modal (ADR-061); write is attach/detach/ toggle-check, also on MCP (ADR-060). Type to get autocomplete from your library; an unknown name creates a new library item. Icons are inline-SVG, never emoji.
