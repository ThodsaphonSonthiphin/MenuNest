State A — ส้ม “Assign every baht” (เลิกเรียก "Ready to Assign") Copy เปลี่ยนเป็น "Assign every baht" เน้น discipline ของ method. มี state icon ⚡ + เลข + cta pill "↑ Still to assign" + progress bar % ของรายรับ + quick chips 2 ตัว:
• Fill targets first = จัดเงินใส่ category ที่มี target ก่อน (รัน priority ตาม TargetDueDate / MonthlyAmount) จนกว่าจะเต็มหรือเงินหมด
• Equally to envelopes = หาร RTA / (จำนวน envelope ที่ยัง funded น้อยกว่า assigned ของเดือนก่อน) เท่า ๆ กัน
กดแล้วเปิด dialog confirm + preview ว่าจะกระจายไปเท่าไหร่ (ใช้ existing SetAssignedAmount mutation วน loop)
