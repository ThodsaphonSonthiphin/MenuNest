# Icons

PNG placeholders for the PWA manifest + push notifications:

- `icon-192.png` — 192x192, used as the notification icon and standalone app icon
- `icon-512.png` — 512x512, large/maskable icon for installers and splash screens
- `badge-72.png` — 72x72, monochrome badge shown in the notification status bar (Android)

All three are flat indigo (#4f46e5) Phase 1 placeholders generated by
`generate-placeholders.cjs`. Re-run with:

```bash
node frontend/public/icons/generate-placeholders.cjs
```

Replace with real logo art (matching theme_color in `manifest.json`)
once design is ready.
